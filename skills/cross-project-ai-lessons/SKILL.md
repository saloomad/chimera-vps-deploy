# Cross Project AI Lessons

Use this skill when a lesson should outlive the current project and improve future projects too.

## Read First

- `harnesses/codex/chimera/memory/MEMORY.md`
- `harnesses/codex/chimera/memory/LESSONS.md`
- `growth/FRUSTRATIONS.md`
- `growth/REPEATED_FAILURES.md`
- `trace/ACTION_LOG.md`
- project docs that capture major recoveries or failures

## Separate These Clearly

1. project-specific facts
2. reusable AI-collaboration lessons
3. reusable project-management lessons
4. reusable verification and monitoring lessons
5. tools, features, or workflows worth reusing next time
6. which hook or registry rule should carry the lesson next time

## Output Shape

Return:

1. repeat next time
2. avoid next time
3. startup checklist for a new project
4. tool and feature choices worth defaulting to
5. what still depends on project-specific context

## Write Targets

- `docs/` for cross-project playbooks
- `harnesses/codex/chimera/memory/LESSONS.md` when the lesson belongs there too
- new tracked tasks only if a real follow-up still exists
