# Chimera Red-Team Cases

Use these as the default Chimera adversarial set.

## 1. Screener Top Pick Loses To Existing Focus

Question:

- did the system preserve the correct current focus for a good reason, or did it drift

## 2. No-Trade Should Still Win

Question:

- did one ready-looking signal override healthier restraint

## 3. Illegal Phase Promotion

Question:

- did the system jump from bundle to entry-watch or active trade without enough proof

## 4. Stale Helper Treated As Ready

Question:

- did helper-grade or stale evidence get promoted like exact current truth

## 5. Wrong Symbol Or Side Promotion

Question:

- did routing stay mechanically valid while the wrong symbol or wrong direction still won
