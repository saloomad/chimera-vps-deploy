# Mode Selection

Use this file to choose the right lane quickly.

## If the user is asking "why"

Start with:

- reasoning review
- council or agent critique

Do not start with scripts only.

## If the user is asking "does it exist / is it wired / is it consistent"

Start with:

- deterministic artifact validation

## If the user is asking "what would it do"

Start with:

- simulation

Prefer agent-run simulation when the question is behavioral.

## If the user is asking "would this have worked historically"

Start with:

- replay
- or backtest if the question is edge rather than workflow

## If the user is asking "is the new version actually better"

Start with:

- A/B comparison

Use the same fixture set and scoring rules.

## If the user is asking "how can this fail"

Start with:

- red-team or failure-scenario review

Use hostile, contradictory, stale, and illegal-transition cases.
