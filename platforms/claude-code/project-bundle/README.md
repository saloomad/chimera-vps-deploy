# Claude Code Chimera Project Hook Bundle

This folder mirrors the Claude Code project-level enforcement bundle used in the Chimera workspace.

What it includes:

- `.claude/settings.template.json`
- hook scripts
- the hook overview note

Why it exists:

- the live `.claude/` folder in the main workspace is intentionally not fully versioned
- this shared mirror gives other platforms and future sessions a pullable source of truth

What it does not claim:

- it does not magically prove every hook is active on every Claude Code install
- it does not replace the live local `.claude/settings.json`
