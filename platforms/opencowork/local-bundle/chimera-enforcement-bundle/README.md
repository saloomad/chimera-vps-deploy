# Chimera Enforcement Bundle

This local OpenCowork plugin adds:

- start and review commands for enforcement-aware work
- local hook guardrails for prompt start, risky tools, proof nudges, and premature stop
- activation receipts written into the Chimera workspace
- carry-forward enforcement for objective status, unapproved items, and remaining work in meaningful replies
- shared GitHub coordination reminders so task changes update session state and publish debt through `github-coordination-gate`, `task-transition-publish`, and `platform-live-repo-router`
